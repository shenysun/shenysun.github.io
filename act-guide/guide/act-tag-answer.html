互动作答数据 ​

选中 tag 下的学生互动作答数据。和互动当前数据一样，这个store也支持传参。不同的参数代表不同的使用场景。

数据来源 ​

学生作答数据

接口：/statistics/v1/lecturer/stu-answer-by-tag

请求方式：GET

请求参数：

json
{
  "interactionId": "互动id",
  "queId": "题目id",
  "queType": "题目类型",
  "tag": "A",
  "result": "CONTENT"
}

返回数据：

json
{
  "actId": "互动id",
  "interactionId": "互动id",
  "tag": "A",
  "result": "CONTENT",
  "total": 10,
  "stuList": [],
  "answer": [
    {
      "stuId": "1",
      "stuName": "张三",
      "puPicUrl": "头像URL",
      "tutorId": "",
      "queId": "题目id",
      "content": "答案",
      "result": "string",
      "score": 0,
      "point": 10,
      "answerTime": null,
      "suggestion": "string",
      "ext": { "ePointCard": null },
      "historyAnswers": null,
      "answerList": null
    }
  ]
}

Pomelo 作答消息

command: TAG_RESULT 和 ABILITY_TAG_RESULT

数据结构 ​

文件地址

packages/act/src/types/modules/server.d.ts

点我查看代码
typescript
export module Details {
  export interface Meta {
    actId: string
    interactionId: string
    tag: string
    result: string
    total: any
    stuList: any
    answer: Array<TagStuData>
  }

  /**
   * tag 作答详情-学生作答信息
   */
  export interface TagStuData {
    stuId: number
    stuName: string
    puPicUrl: string
    tutorId: string
    queId: string
    content: string
    result: string
    score: number
    point: number
    answerTime: number
    suggestion: string
    ext: { ePointCard: null }
    historyAnswers: Array<any>
    answerList: null
  }
}

store api ​

文件地址

packages/act/src/stores/modules/act-tag-answer.ts

currentTagTable ​

当前 tag 作答数据。数据类型是 ActClient.Details.TagStuData[]。

currentTag ​

当前选中 tag。数据类型是 ActClient.Details.ShowDetailsTag。

tagList ​

tag 列表。数据类型是 ActClient.Details.ShowDetailsTag[]。

selectedStuList ​

选中学生列表。数据类型是 ActServer.Details.TagStuData[]。

setCurrentTag ​

设置当前 tag。

setSelectedStuList ​

增删选中学生列表。

updateCurrentTagDetails ​

更新 tag 作答数据。数据来源Pomelo和接口。
