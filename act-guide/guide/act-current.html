互动当前数据 ​

获取当前互动数据。当前互动可能是学情历史互动、或者是当前正在发起的互动。所以这个store支持入参：

  • single 单个互动预览，一般用于学情
  • total 当前最新的一批互动数据，一般用于互动回显
  • ctrl 互动控制数据，一般用于互动回显

数据来源 ​

数据的组合，主要从互动物料和互动统计数据中获取。

export class ActBasic {
  actId: string = '';
  queIndex: number = 0;
  actStatistics = useActStatistics();
  materialStore = useMaterialStore();

  constructor(id: string, queIndex: number = 0) {
    this.actId = id;
    this.queIndex = queIndex;
  }

  get currentAct() {
    return this.actStatistics.getStatistic(this.actId);
  }

  get material() {
    return this.materialStore.getActMaterial(this.currentAct?.actTempId);
  }

  get currentTask() {
    return this.material?.tasks.find(t => t.queBody.queId === this.queId);
  }

  get currentQue() {
    return this.currentAct?.questionList.find(q => q.qId === this.queId);
  }

  get queId() {
    return this.currentAct?.questionList[this.queIndex]?.qId;
  }

  setActId(id: string) {
    this.actId = id;
  }

  setQueId(queId: string | number) {
    if (typeof queId === 'number')
      this.queIndex = queId;
    else
      this.queIndex = this.currentAct?.questionList.findIndex(q => q.qId === queId) ?? 0;
  }
}

store api ​

currentAct ​

当前互动数据。数据类型是 StandardStatistic。

material ​

当前互动物料数据。数据类型是 ActServer.Material.Meta。

currentTask ​

当前物料题目数据。数据类型是 ActServer.Material.TaskData。

currentQue ​

当前互动题目作答数据。数据类型是 ActClient.Statistic.QuestionInfo。

queId ​

当前互动题目id。

setActId ​

设置当前互动id。

setQueId ​

设置当前互动题目id。
